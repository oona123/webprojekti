Taulukko

Aihe Kuvaus Käyttötarkoitus
HTML HyperText Markup Language (HTML) on merkintäkieli, jota käytetään verkkosivustojen rakenteen ja sisällön määrittämiseen. Verkkosivujen rakenne ja sisältö, kuten otsikot, kappaleet, listat, kuvat ja lomakkeet.
CSS Cascading Style Sheets (CSS) on tyyliarkkikieli, jolla määritellään verkkosivujen ulkoasu, kuten värit, fontit ja layoutit. Verkkosivujen ulkoasun ja tyylin muokkaaminen.
JavaScript JavaScript on ohjelmointikieli, joka mahdollistaa verkkosivujen dynaamisen ja vuorovaikutteisen käyttäytymisen. Käytetään interaktiivisten elementtien luomiseen.
Asynkroniset Operaatiot Asynkroniset operaatiot mahdollistavat koodin suorittamisen taustalla. Käytetään esimerkiksi tietojen hakemiseen palvelimelta.
